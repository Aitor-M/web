Old Town Stories

Old Town Stories is a VR (Oculus Rift) shooter game made with Unreal Engine 4. We were a group of 10 students, 6 programmers and 4 artists, that worked for 8 months to complete the demo which features a tutorial, a story-mode and an arena.

  • VR integration

    The player's Pawn. VR related input and hand animations integration.

  • VR optimization

    Making sure the project had a stable 90fps was a challenge due to the realistic look of the game. Some of the optimizations were: making sure there were no transparencies, level-streaming, using as simple as posible materials, HLODS, static lights...

  • Magnetic Shield

    Shield which stops, accumulates and returns enemy bullets.

  • VFX

    Shield which stops, accumulates and returns enemy bullets.

  • Tutorial

    Event based tutorial which triggers new actions everytime the player interacts with the world.

GNX Engine

GNX Engine is a multiplatform (Windows/Linux) OpenGL 3D engine featuring a multithreaded workflow using Tasks and Render Queues, as well as, scripting (AngelScript), FXAA, Bloom and HDR.

Gravity Shifter

Gravity Shifter is a 2D platformer game based on gravity, movement fluidity and slow motion.

The Challenge

Gravity Shifter is played by using just the two joysticks on the controller. (even the menus!). It was a design request made by my university teacher @SuperLemonBits!
